# Game Tes Mengetik - Kalimat Acak Tak Terbatas

Game tes mengetik interaktif dengan sistem kalimat acak tak terbatas dalam waktu 60 detik.

## Fitur Utama

### 1. **Sistem Kalimat Acak**
- 100+ kalimat dalam 3 level kesulitan
- Kalimat terdiri dari 1-2 baris teks
- Database kata yang kaya dan bervariasi
- Mode Campuran untuk variasi maksimal

### 2. **Timer 60 Detik**
- Ketik sebanyak-banyaknya dalam 1 menit
- Auto-start saat mulai mengetik
- Countdown dengan feedback visual

### 3. **Highlight Real-time**
- Karakter benar: Hijau dengan highlight
- Karakter salah: Merah dengan coretan
- Karakter saat ini: Biru berkedip
- Auto-scroll ke posisi saat ini

### 4. **Statistik Lengkap**
- **KPM (Kata per Menit)**: Kecepatan mengetik
- **Akurasi Baris**: Persentase baris benar
- **Skor Total**: Perhitungan komprehensif
- **Baris Benar/Salah**: Statistik detail

### 5. **Fitur Tambahan**
- **4 Mode Kesulitan**: Pendek, Sedang, Panjang, Campuran
- **Skip Baris**: Lewati kalimat yang sulit
- **Clear Input**: Bersihkan teks yang sedang diketik
- **Sound Effects**: Feedback audio untuk aksi
- **Riwayat Baris**: Track progress terakhir
- **Rating Bintang**: Evaluasi performa

### 6. **UI/UX Modern**
- Responsive design untuk semua device
- Dark mode yang eye-friendly
- Animasi smooth dan interaktif
- Visual feedback yang jelas

## Cara Menggunakan

### Instalasi
1. Download semua file (index.html, style.css, game.js)
2. Simpan dalam folder yang sama
3. Buka index.html di browser

### Cara Bermain
1. **Pilih Mode**: Pilih kesulitan kalimat
2. **Mulai Game**: Klik "Mulai Game" atau langsung mengetik
3. **Ketik Kalimat**: Ketik ulang kalimat yang muncul
4. **Submit**: Tekan ENTER untuk cek dan lanjut
5. **Ulangi**: Ketik sebanyak-banyaknya dalam 60 detik
6. **Lihat Hasil**: Skor dan rating akan ditampilkan

### Kontrol
- **ENTER**: Submit teks dan lanjut ke kalimat baru
- **Ctrl+S**: Skip kalimat saat ini
- **Ctrl+L**: Clear input field
- **Space**: Normal typing (tidak ada fungsi khusus)
